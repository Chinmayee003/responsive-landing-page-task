# Responsive Landing Page

## Objective
Built a clean and responsive landing page using HTML and CSS.

## Features
- Header with logo and navigation
- Hero section with heading, paragraph and CTA button
- Responsive navigation (collapses on small screens)
- Footer with social links
- Mobile responsive using media queries

## Tools Used
- HTML5
- CSS3 (Flexbox, Media Queries)
- VS Code
- Chrome Browser
- Live Server

## How to Run
Open index.html in browser or run using Live Server.
